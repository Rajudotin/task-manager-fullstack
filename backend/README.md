# Task Manager API - Backend (MySQL)

A scalable REST API with authentication and role-based access control for task management using MySQL.

## Features

- User registration & login with JWT authentication
- Role-based access (user vs admin)
- CRUD operations for tasks
- API versioning, error handling, validation
- Swagger API documentation
- MySQL database with Sequelize ORM
- Security features (helmet, rate limiting, sanitization)
- Logging with Winston
- Scalable project structure

## Tech Stack

- Node.js & Express.js
- MySQL with Sequelize ORM
- JWT for authentication
- Bcrypt for password hashing
- Winston for logging
- Swagger for API documentation

## Prerequisites

- Node.js (v14 or higher)
- MySQL (v5.7 or higher)
- npm or yarn
